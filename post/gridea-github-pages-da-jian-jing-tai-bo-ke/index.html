Gridea + Github-Pages搭建个人博客

· 2024-03-21 · # 技术类 # Github-Pages # Gridea

1、为什么选择Gridea

Wordpress：什么都有，唯一的毛病比较慢。
Ghost：生态不错，但考虑到国内的网络环境和本地化程度，只适合做英文站。
Hexo：折腾型选手，缺点是文章过多编译时太慢。
Hugo：折腾型选手，编译效率上去了，但是写文章体验不是很好。
Gridea：有独立客户端，几乎没有生态，但用起来很方便，适合专注创作的朋友。

Gridea作为一个小而美的静态博客写作客户端，可以让你将精力专注于写作上面，操作方便快捷，无需Cmd执行Git命令就可一键同步。

2、为什么选择Github-Pages

GitHub-Pages的优点是免费、简单易用、稳定可靠，缺点是不支持动态网站和数据库，只能托管静态网站，对于个人博客而言已经够用。而且在知识商业化、知识产权等方面，相比于国内各大博客平台（CSDN之类的），Github-Pages显然更值得信任。
